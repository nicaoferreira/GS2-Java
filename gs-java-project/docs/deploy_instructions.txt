Deploy no Render - Passo a passo

1) Crie repositório no GitHub e faça push de todo o projeto.
2) No Render.com, "New" -> "Web Service" -> conecte ao GitHub e escolha o branch principal.
3) Build Command: mvn clean package
   Start Command: java -jar target/gs-java-project-0.0.1-SNAPSHOT.jar
4) Environment variables to set on Render:
   - SPRING_DATASOURCE_URL=jdbc:oracle:thin:@<HOST>:<PORT>/<SERVICE_NAME>
   - SPRING_DATASOURCE_USERNAME=<DB_USER>
   - SPRING_DATASOURCE_PASSWORD=<DB_PASS>
   - SPRING_PROFILES_ACTIVE=prod
